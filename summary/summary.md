- 快慢指针：如果链表中节点个数为偶数时，当快指针无法继续移动时，慢指针刚好指向中点；如果链表中节点个数为奇数时，当快指针走完，慢指针指向中点前一个节点。

    ```cpp
    while (fast->next && fast->next->next) {
        slow=slow->next;//slow指针每次走一步
        fast=fast->next->next;//fast指针每次走两步
    }
    ```
